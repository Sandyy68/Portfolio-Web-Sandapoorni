# Portfolio-Web-Sandapoorni

## 📌 Overview
This is my personal portfolio website, designed to showcase my skills, projects, and professional experience. The site is built with **HTML5, CSS3, JavaScript, PHP, and MySQL**, ensuring a responsive and engaging user experience.

## 🚀 Features
- **Modern & Responsive Design** – Optimized for both desktop and mobile users.
- **Project Showcase** – Displays key projects with detailed descriptions.
- **Contact Section** – Allows visitors to get in touch easily.
- **Dynamic Content** – Integrates backend functionality using PHP and MySQL.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP, MySQL
- **Version Control:** Git, GitHub
